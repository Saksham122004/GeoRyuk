# GeoRyuk

GeoRyuk is a simple Python scraper for Google Maps listings using Selenium.

## Features

- Scrapes business listings from Google Maps
- Extracts name, rating, address, phone, and website
- Saves results to a CSV file

## Requirements

Install dependencies with:

```bash
pip install -r requirements.txt
```

## Usage

Run the script:

```bash
python GeoRyuk.py
```

Then enter the search query and number of results when prompted.
